WEB ÔN THI TRẮC NGHIỆM NNLT (offline)

1) Cách chạy nhanh nhất:
- Mở file index.html bằng Chrome/Edge/Firefox.

2) Nếu trình duyệt chặn file cục bộ:
- Cách 1 (khuyên dùng): mở bằng Chrome/Edge.
- Cách 2: chạy server tĩnh (nếu bạn biết dùng):
  Python:  python -m http.server 8000
  Sau đó vào: http://localhost:8000

Tính năng:
- Chọn chương, số câu, chế độ Luyện tập / Thi thử
- Trộn câu / trộn đáp án
- Đánh dấu câu, nhảy nhanh theo số câu
- Kết quả theo chương, làm lại câu sai
- Lưu lịch sử & điểm best bằng localStorage (trên cùng trình duyệt)
